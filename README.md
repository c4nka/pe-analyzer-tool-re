## Proje Hakkında
Bu araç, Windows PE (Portable Executable) dosyalarının statik analizini yapmak ve entropi (Shannon Entropy) algoritması kullanarak dosyaların paketlenmiş (packed) veya şifrelenmiş olma ihtimalini tespit etmek amacıyla geliştirilmektedir.
