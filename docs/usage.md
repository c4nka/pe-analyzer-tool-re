# Kurulum ve Kullanım

## Ön Koşullar
* Python 3.x
* Git

## Kurulum
1. Depoyu klonlayın ve proje dizinine gidin.
2. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install -r requirements.txt

Uygulamayı çalıştırmak için terminal üzerinden analiz edilecek .exe veya .dll dosyasının yolunu argüman olarak vermelisiniz:

```bash
python main.py <dosya_yolu>
```

Örnek

```bash
python main.py C:\Windows\System32\calc.exe
```
